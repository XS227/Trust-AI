Shopify integration

Setting up with Shopify

Two steps to connect your Shopify store. No app install required — a lightweight script and one webhook.

Step 1 — Add the tracking script to theme.liquid

In Shopify Admin → Online Store → Themes → Edit Code → layout/theme.liquid, paste before </body>:

<script src="https://trustai.no/api/trustai-referral.js?v=2"></script>

Step 2 — Add an order webhook

In Shopify Admin → Settings → Notifications → Webhooks, create an orders/create webhook pointing to:

https://trustai.no/api/shopify-order-webhook.php

Once both are in place the system automatically tracks clicks, orders and commissions.

For ambassadors

Apply at ambassador-signup.html

New ambassadors fill in the application form and select which store they want to represent.

2

Merchant approves the application

Once approved, the ambassador receives access and their unique referral link is activated.

3

Log in and share

Ambassadors log in at login.html, copy their referral link and start sharing across SMS, social media or WhatsApp.

Automatic commission tracking

Every click is tracked. When a referred customer completes a purchase, the commission is calculated automatically and appears in the ambassador's dashboard.

FAQ

Common questions

How many ambassadors can we have?

There is no limit. The platform is built to scale with your store.

Can we use TrustAI across multiple stores?

Yes. Each Shopify store can be connected with its own setup, ambassadors and commission rules.

Are there limits on referral links or clicks?

No. Ambassadors can share freely and the system tracks automatically.

How quickly does data update?

Clicks and order data update in real time. Dashboards reflect activity immediately.

How much technical work is required?

Very little. You add the tracking script and webhook once. Everything else runs automatically.
