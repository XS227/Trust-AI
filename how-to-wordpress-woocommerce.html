Manuell installasjon (midlertidig)

Hvis plugin ikke er aktiv enda, kan scriptet legges inn midlertidig:

<script src="https://trustai.no/api/trustai-referral.js"></script>

For WordPress bør scripts normalt legges inn med wp_enqueue_script(), via wp_enqueue_scripts-hooken, i stedet for hardkoding i theme header.

WooCommerce ordre-sporing

Pluginen kan sende ordre til TrustAI med WooCommerce hooks eller webhooks. Vanlig oppsett er hook etter checkout (f.eks. woocommerce_thankyou) og/eller webhooks ved ordre-hendelser.

FAQ

Må kunden kunne kode?

Nei, plugin-oppsettet er laget for ikke-tekniske brukere.

Fungerer dette med alle WordPress themes?

Ja, så lenge temaet følger standard WordPress hooks.

Hva skjer hvis kunden betaler senere?

Salg registreres når ordrehendelsen sendes til TrustAI.

Hvor lenge varer referral-cookie?

Butikken velger cookie-varighet i TrustAI-oppsettet.

Kan man bruke dette uten WooCommerce?

Ja, TrustAI kan brukes med andre plattformer og manuell integrasjon.

Kan TrustAI brukes for leads, ikke bare salg?

Ja, TrustAI kan også spore leads og andre konverteringer.
