Installer TrustAI på Wix

Spor ambassadørlenker, leads og konverteringer fra Wix-nettsiden din – uten tung teknisk setup.

Start gratis Se installasjonsguide

Klar på få minutter

Legg inn én script-tag i Wix Custom Code, publiser, og start sporing av referrals og leads.

Wix-oppsett: Settings → Custom Code → Add Custom Code → All pages

Slik installerer du på Wix

  1. 1. Logg inn på Wix Dashboard.
  2. 2. Gå til Settings → Custom Code.
  3. 3. Klikk Add Custom Code.
  4. 4. Lim inn scriptet:
<script src="https://trustai.no/api/trustai-referral.js"></script>
  1. 5. Velg All pages.
  2. 6. Velg plassering Body - end eller Head.
  3. 7. Trykk Apply og deretter Publish.

Test installasjonen

Åpne nettsiden din med en testlenke, for eksempel:

https://dinside.no/?ref=TEST123

Deretter sjekker du TrustAI-dashboardet for registrerte klikk og referral.

Viktig om Wix checkout

Wix checkout kan ha begrensninger sammenlignet med Shopify. TrustAI fungerer best på Wix når målet er leads, kontaktskjema, booking eller forespørsler. Full kjøpssporing kan kreve ekstra oppsett med Velo eller Wix events.

For avansert oppsett

  • Velo by Wix kan brukes for mer avansert tracking.
  • Backend logic bør ligge sikkert server-side.
  • TrustAI kan kobles mot custom events senere.

FAQ

Må jeg kode?
Nei. For standardoppsett holder det å lime inn scriptet i Wix Custom Code.

Fungerer dette på alle Wix-sider?
Ja, velg “All pages” for å aktivere scriptet på hele nettstedet.

Kan jeg bruke TrustAI for bookings?
Ja. Bookinger og forespørsler kan spores som leads.

Kan ambassadører få dashboard?
Ja, de får eget dashboard med lenker, klikk og resultater.

Hva hvis jeg bruker Wix Stores?
Grunnsporing fungerer, men full kjøpssporing i checkout kan kreve ekstra Velo/Wix events-oppsett.
